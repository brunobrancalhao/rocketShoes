## Configuração

1. Instale todas as dependências:

    ```sh
    $ npm install
    ```
  
2. Inicie o server:

    ```sh
    $ npx json-server server.json -p 3333
    ```
  
3. Rode o projeto:

    ```sh
    $ npm start
    ```
